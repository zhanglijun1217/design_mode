# design_mode
- 设计模式学习，用一些例子去学习设计模式，每个package下还会有对应的一个笔记pdf。尽量做到至少一星期学习一个设计模式，目前计划是23中设计模式。
- 对应每个设计模式会写一个博客进行总结
- package daily_tools 用来记录一些在工作中遇到的设计模式的应用。设计模式我理解的是一个工具，用多了才能在遇到新的场景时选择扩展性高、维护性高的设计模式
